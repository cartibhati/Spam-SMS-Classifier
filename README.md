# 📩 SMS Spam Classifier

A machine learning project that classifies SMS messages as **Spam** or **Not Spam (Ham)** using Natural Language Processing techniques.

---

## 🚀 Features

* Text preprocessing (cleaning, tokenization)
* TF-IDF vectorization
* Machine Learning model (Naive Bayes)
* Real-time prediction support

---

## 🧠 Tech Stack

* Python
* Scikit-learn
* Pandas
* NumPy


---

## 📂 Project Structure

```
SMS_Spam_Classifier/
│
├── app.py              # Application logic
├── main.py             # Entry point
├── vectorizer.pkl      # TF-IDF vectorizer
├── model.pkl           # Trained ML model
├── .gitignore
```

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```
